What should the center bar primarily do?

VS Code's center bar started as a command palette trigger. We have design freedom to define ours. Pick the primary role:

A

Command Palette

Run any editor action — open settings, toggle grid, change tool, switch layout, etc. Like VS Code's Ctrl+Shift+P. Great for keyboard-first power users and AI agent automation.

> Toggle Grid
> Open Project…
> Set Tool: Move
B

Scene Search / Navigation

Jump to entities, assets, components by name. Select and focus. Like a scene-scoped quick-open. Great for large scenes with many entities.

@ Player
@ EnemySpawner_01
@ MainCamera
C

Unified Omnibar

One input, everything. Prefix characters route to different modes — > for commands, @ for entities, # for assets, no prefix = fuzzy search all. Exactly what VS Code does with its search bar in newer versions.

> commands   @ entities   # assets   ? help
