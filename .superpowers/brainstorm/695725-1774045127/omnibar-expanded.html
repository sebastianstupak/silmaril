Interaction rules

  • Ctrl+K — open omnibar from anywhere
  • Arrow keys — navigate results
  • Enter — execute selected, dismiss
  • Escape — dismiss, return focus to previous element
  • Tab — cycle between result groups
  • Prefix > — commands only   @ — entities only   # — assets only
  • Empty input — shows recent projects/scenes + most-used commands
  • Clicking outside or pressing Escape — dismisses with no action
