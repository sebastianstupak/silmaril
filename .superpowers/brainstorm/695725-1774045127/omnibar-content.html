What does the omnibar surface?

When you type, what should appear in the dropdown? Select all that apply.

1

Editor Commands

Run any action: Toggle Grid, Open Settings, Set Tool: Move, Switch Layout, New Scene… — like VS Code's > prefix. The power-user backbone.

2

Scene Entities

Find and select/focus entities by name: Player, EnemySpawner_01, MainCamera… — equivalent to VS Code's @ symbol jump. Essential for large scenes.

3

Keybind Discovery

Show shortcuts inline: "Toggle Grid — Ctrl+G", "Play — F5". Doubles as a help system — you can find the action and learn the shortcut at the same time.

4

Recent Projects / Scenes

Quick-switch: Open recent project, load a saved scene. Appears when search is empty — like VS Code's recent files on empty Ctrl+P.

5

Assets

Find textures, meshes, scripts, prefabs by name when a project is open. Like VS Code's file quick-open but for game assets.

You can select multiple. The more you select, the richer (and more complex) the results view becomes.
