A — VSCode Tree

Indented folder tree, expand/collapse, single file selected. Right-click context menu for new file/folder, rename, delete. Familiar to any VSCode user.

B — Miller Columns

Two-pane column view. Left shows parent folder contents, right shows selected folder contents. Breadcrumb at top. Good for wide exploration but tight in narrow panels.

C — Flat List + Breadcrumb

Shows one folder at a time. Breadcrumb header to navigate up. Folders and files separated by divider. Simpler, works well in narrow panels. Less overview at a glance.
