## 📚 **Documentation Standards**

Each `.md` file should have:

1. **Title and description**
2. **Table of contents** (for files >500 lines)
3. **Code examples** (must compile)
4. **Diagrams** (text-based, ASCII art)
5. **Related docs** (links at bottom)
6. **Last updated date**
